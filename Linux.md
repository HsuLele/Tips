🗂️ 一、目录与文件操作

| 功能         | 命令                     | 说明                                  |
| ------------ | ------------------------ | ------------------------------------- |
| 显示当前路径 | `pwd`                    | print working directory               |
| 查看目录内容 | `ls` / `ls -l` / `ls -a` | 列出文件、显示详细信息、包含隐藏文件  |
| 切换目录     | `cd 目录名`              | 例：`cd /home/user`，`cd ..` 返回上级 |
| 创建目录     | `mkdir 目录名`           | 例：`mkdir test`                      |
| 删除目录     | `rmdir 目录名`           | 仅删除空目录                          |
| 递归删除目录 | `rm -rf 目录名`          | 删除整个目录及内容（⚠危险操作）       |
| 创建文件     | `touch 文件名`           | 例：`touch a.txt`                     |
| 复制文件     | `cp 源 目标`             | 例：`cp a.txt /tmp/`                  |
| 移动/重命名  | `mv 源 目标`             | 例：`mv old.txt new.txt`              |
| 删除文件     | `rm 文件名`              | 例：`rm a.txt`                        |
| 查看文件内容 | `cat` / `more` / `less`  | `less` 可翻页查看                     |
| 搜索文件     | `find 路径 -name 文件名` | 例：`find / -name "*.log"`            |

## 🧰 一、通过 `scp`（最常用）

`scp` = **secure copy**，基于 SSH 安全通道复制文件。

### 📤 从本地上传到远程服务器

```
scp 本地文件 用户名@服务器IP:远程路径
```

**示例：**

```
scp test.txt root@192.168.1.10:/home/root/
```

意思是：把本地 `test.txt` 上传到远程 `/home/root/` 目录。

------

### 📥 从远程服务器下载到本地

```
scp 用户名@服务器IP:远程文件路径 本地路径
```

**示例：**

```
scp root@192.168.1.10:/home/root/test.txt /Users/lele/Desktop/
```

将远程文件下载到本地桌面。

------

### 📁 上传/下载整个目录

加上 `-r` 递归选项：

```
scp -r 本地目录 用户名@IP:远程目录
scp -r 用户名@IP:远程目录 本地目录
```
